# Agent 的定义

## 概述

在人工智能的经典定义中，Agent（代理）被描述为一个“感知其环境、推理其状态和目标，并采取行动来实现目标”的系统，且通常以迭代循环的方式运行 1。这种循环机制使其能够在一个动态环境中工作，并根据新的观察结果调整其行为。

当这一概念与大型语言模型（LLM）结合时，即形成了 LLM Agent。在这种架构下，“感知”是通过自然语言理解和处理输入信息实现的，而“行动”则是通过工具使用、API 调用或与外部系统的交互来完成 1。Agent 的核心价值在于其**自主规划**能力 2。与需要精确输入和固定流程的传统软件不同，AI Agent 致力于确定实现所需复杂目标的步骤，能够在缺乏清晰指令的情况下动态提出解决方案，甚至通过试错来学习 2。
